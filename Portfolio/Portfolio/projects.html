Blue Orchid Hotel

Take a look at !

The 'Hotel Blue Orchid' website was conceived as a complete web application, designed following the Model-View-Controller (MVC) architectural pattern. For the backend logic, Java was implemented using the Spring Boot framework. On the frontend, HTML, CSS, and the Bootstrap framework were combined to provide an attractive and responsive user interface, ensuring an optimal experience across various devices. Data persistence was achieved through the use of SQL in MySQL Workbench, configuring a relational database to store relevant hotel and customer information



VideoGame Tournament

Take a look at

Developed as a final project for the Data Structures and Algorithms course, 'KO VideoGame Tournament' is a video game tournament simulation that leverages the power of data structures and algorithms. Specifically, linked lists were used for participant management, binary trees for organizing elimination rounds, and sorting algorithms for ranking players based on their performance. This project demonstrates the practical application of theoretical concepts in a competitive event simulation context
